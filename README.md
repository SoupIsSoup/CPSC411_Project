# FoodFinder

FoodFinder is an iOS restaurant search app built with Swift and Xcode. 
It uses the Yelp API to search for restaurants by location and category.

## Features
- Search restaurants by location
- Filter by category
- View restaurant details
- Save favorites locally

## Technologies
- Swift
- SwiftUI
- Xcode
- Yelp API
- REST API
- JSON / Codable
- SQLite (local persistence)

## How To Run
- Use Xcode 15+ and its IOS Simulator
- Install the following dependencies:
  - SQLITE https://github.com/stephencelis/SQLite.swift
  - Yelp API (put the real Secrets.swift file in FoodFinder/Utils)
- Run app
